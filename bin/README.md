克隆
git clone https://github.com/air459/Air-Tool.git
mv Air-Tool 安装路径
nano .bashrc
找一行添加
export Air-Tool="sh /xx/xx/Air-Tool/bin/main.sh" 或 alias Air-Tool="sh /xx/xx/Air-Tool/bin/main.sh"
安装Termux-api
[github](https://github.com/termux/termux-api)
